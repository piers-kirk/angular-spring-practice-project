This Application

I developed this application with the aim of enhancing my skills in Angular and Spring Boot. The backend is written using Java 8 with Spring Boot. I am using the Apache POI library to create the Show Export. The frontend is written using Angular. I am using Docker to host a Microsoft SQL Server instance.
